# STM32 Development Toolchain (In progress)

This project serves as a skeleton to develop applications for the different stm32 devices

## Features
* Dependency manager (STMCube)
* Static Analysis Tools
..

## How to use

Dependencies:
* CMake
* arm-none-eabi-gcc
* doxygen(optional)
* cppcheck(optional)
* clang(optional)
* ...
